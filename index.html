Control Pongbot Nova S Pro

  1. Make sure Bluetooth is enabled on your device. (Wifi or mobile data is not necessary after loading this page)
  2. Click button and select your Nova-Bot
  3. After a connection could be established, your bot should no longer show blue blinking lights but rather keep the blue lights on continuously.
  4. Change the drill configuration in the text area below (according to the valid parameter values given in the list below).
  5. Click button "Run drill" to send the drill to the bot and start the drill
  6. Either use buttons "Pause", "Continue", "Stop" as you see fit.
  7. Save the contents of the text area (drill definition) to your local device via copy and paste or the buttons at the bottom.

Drill definition and control

One line per ball, or multiple balls per line separated by | (vertical bar) to let the bot select one of multiple balls at random (for example serve first, then randomly drive fh or bh. There can be as many lines as you like.

Comma or spaces can be used to separate numbers. A semikolon character ';' means the rest of the line is ignored, useful for comments.

  1. speed upper wheel 500 <=x <= 7200 rpm (step size 1)
  2. speed lower wheel 500 <=x <= 7200 rpm (step size 1)
  3. ball height -50 <=x <= 100 (step size 1)
  4. drop point -10 <=x <= 10 (step size 0.5)
  5. frequency 0 <=x <= 100 (step size 10)
  6. repetitions 1 <=x <= 200 (step size 1)
